ASCII Box Demo

This page demonstrates how to use the ASCII Box utilities in the Mountain of Many Voices project.

Interactive Demo

Usage Instructions

1. Import the utility

import { createAsciiBox, applyAsciiBoxesToClass, createAsciiBoxButton } from '/sites/shared/ascii-box-utils.js';

2. Use in your code

// Create an ASCII box as a string
const boxedText = await createAsciiBox('Hello World', {
  border: 'double',
  padding: 1
});

// Apply to a button element
const button = document.querySelector('#my-button');
await createAsciiBoxButton(button, {
  border: 'single',
  color: 'blue'
});

// Apply to all elements with a specific class
await applyAsciiBoxesToClass('ascii-box-element', {
  border: 'round',
  color: 'green',
  padding: 1
});

3. Available options

  • border: 'single', 'double', 'code', 'round', 'dotted', 'retro', 'single-double', 'double-single'
  • color: 'green', 'red', 'blue', 'cyan', 'magenta', 'gray', 'black', 'white'
  • padding: Number of spaces inside the box
  • minWidth: Minimum width of the box
  • maxWidth: Maximum width of the box
